Key Learnings

Understanding Bash fundamentals
I learned how to use core Bash concepts such as variables, conditionals (if statements), loops, and functions. This helped me build scripts that can take user input, make decisions, and automate tasks.


File and directory operations
I gained confidence working with files and directories using commands like mkdir, cd, cp, cat, and file test operators such as -f, -r.


Input validation and error handling
I learned how to safely handle errors, such as preventing division by zero and checking if a file exists before performing operations. Using checks like [ "$num2" -ne 0 ] and Bash options like set -u helped make scripts more reliable.
